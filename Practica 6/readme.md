## Часть 2: Настройка туннеля GRE

### Шаг 1: Настройка туннеля GRE на маршрутизаторе RA

enable

conf t

int tunnel 0

ip address 10.10.10.1 255.255.255.252

tunnel source s0/0/0

tunnel destination 209.165.122.2

tunnel mode gre ip
